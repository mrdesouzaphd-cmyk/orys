## 🎯 ACTIVATION FAMILY (Camada 1 — Ativação)

### Template 1 — 🧭 Trilha de Ativação *(Activation Trail)*

**Purpose:** Open class with a 3-phase activation: independent thinking → paired comparison → consensus share.

**Camada:** 1 (Ativação)
**Bloom level:** Understand → Apply → Analyze
**Time budget:** 5–10 min (min 3, max 15)
**Counteracts:** Orys-2, Orys-3, Orys-10

**Canonical structure:**

```

[Camada 1 — Orys Learning Architecture™ · ~7 min]

═══════════════════════════════════════════════
PERGUNTA DISPARADORA / TRIGGER QUESTION
[1 strong question naming the cognitive operation]
═══════════════════════════════════════════════

┌─────────────────────────────────────────────┐
│ FASE 1 — EU PENSO (~2 min, silêncio)        │
│ Responda individualmente, sem conversar.    │
│                                              │
│ [large empty box]                            │
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│ FASE 2 — MEU COLEGA PENSA (~3 min, em dupla)│
│ Compare. O que é igual? O que é diferente?  │
│                                              │
│ [large empty box]                            │

┌─────────────────────────────────────────────┐
│ FASE 3 — VAMOS COMPARTILHAR (~2 min)        │

Citação: Lyman (1981); Ausubel (1968); de Souza (2025).
```

## 🔍 TEXT ANALYSIS FAMILY (Camada 2 — Estratégias Bloom)

### Template 2 — 🔍 Decomposição Estrutural por Parágrafo *(Structural Decomposition by Paragraph)*

**Purpose:** For each paragraph of a reading: identify central concept + supporting evidence + theoretical lens.

**Camada:** 2
**Bloom level:** Analyze
**Time budget:** 20–30 min (min 15, max 45)
**Counteracts:** Orys-2, Orys-4

**Canonical structure:** Table with one row per paragraph, three columns:
- **Conceito central** (central concept)
- **Evidência de apoio** (supporting evidence — direct quote or paraphrase + location)
- **Lente teórica relevante** (relevant theoretical lens — Ausubel / Goleman / Dweck / domain author)

Bottom of page: **Síntese sistêmica** — one paragraph integrating all concepts identified.

Citation: Anderson & Krathwohl (2001); de Souza (2025).

## ⚖️ CRITICAL ANALYSIS FAMILY (Camada 2 — advanced)

### Template 5 — ⚖️ Análise Crítica ACE *(ACE Critical Analysis)*

**Purpose:** Discipline-agnostic source/argument analysis. ACE = **A**uthor + **C**ontext + **E**vidence.

**Camada:** 2
**Bloom level:** Analyze → Evaluate
**Time budget:** 25–40 min (min 20, max 60)
**Counteracts:** Orys-2, Orys-4

**Canonical structure:** 3-column table:
- **A — Autor / Author**: Who produced this? Credentials, perspective, possible biases.
- **C — Contexto / Context**: When, where, why, for whom was this produced?
- **E — Evidência / Evidence**: What claims are made? What evidence supports them? Are there gaps?

Below the table: **Síntese crítica** — your evaluation of the source's strength and limitations + how it relates to other sources you've encountered.

Note: ACE works for ANY discipline (Diffit's HIPP is locked to AP US History; ACE is universal).

Citation: Wineburg (1991) — Historical Thinking adapted; Toulmin (1958) — Argument Structure; de Souza (2025).

**Purpose:** Quick 5-criteria × 3-level rubric for formative evidence.

**Camada:** 4
**Bloom level:** Evaluate (instructor + student self-assessment)
**Time budget:** 10 min (min 5, max 15)
**Counteracts:** Orys-4, Orys-5

**Canonical structure:** Standard 3-level rubric — see `orys-learning-architecture.md` for the canonical form. 5 rows × 3 columns (Suficiente / Parcial / Insuficiente). Always includes one criterion tagged to the active Orys Pattern™ if one is identified.

Citation: Black & Wiliam (1998); Sadler (1989); de Souza (2025).

### Template 7 — 💬 TAR — Feedback entre Pares *(TAR Peer Feedback)*

**Purpose:** Original Orys peer-revision protocol. TAR = **T**recho marcante + **A**rgumento questionado + **R**ecomendação acionável.

**Camada:** 4 (revision cycle)
**Bloom level:** Analyze → Evaluate
**Time budget:** 20–30 min (min 15, max 45)
**Counteracts:** Orys-4, Orys-5, Orys-8

**Canonical structure:**

```
💬 TAR — Feedback entre Pares
DR. DE.SOUZ.AI Orys™ — Camada 4 — Orys Learning Architecture™

ETAPA 1 — Eu respondo a pergunta:
[Question goes here]
[large answer box]

ETAPA 2 — Troco com meu colega. Leio o trabalho dele e devolvo:

┌─────────────────┬─────────────────┬─────────────────┐
│ T - TRECHO      │ A - ARGUMENTO   │ R - RECOMENDAÇÃO│
│ Marque um       │ Questione um    │ Sugira uma ação │
│ trecho forte    │ argumento que   │ específica que  │
│ que se          │ precisa de mais │ melhoraria o    │
│ destacou.       │ evidência.      │ trabalho.       │
├─────────────────┼─────────────────┼─────────────────┤
│                 │                 │                 │
│                 │                 │                 │
└─────────────────┴─────────────────┴─────────────────┘

ETAPA 3 — Reviso meu trabalho com o feedback recebido:
[large revised-answer box]

Citação: Brookhart (2013); Hattie & Timperley (2007); de Souza (2025).
```

**Why TAR is original (vs. K-12 alternatives):** TAR demands a *marked passage*, a *challenged argument*, and an *actionable recommendation* — three concrete cognitive operations, not vague "tell me something I liked." Designed for higher-ed and graduate work.

## 🛠️ PROCEDURAL TRAINING FAMILY (Camada 5 — Treino Procedimental)

### Template 8 — 🛠️ Roteiro de Simulação Procedimental *(Procedural Simulation Script)*

**Purpose:** Structured 5-step procedural training script for any practical/clinical/lab activity.

**Camada:** 5
**Bloom level:** Apply → Analyze
**Time budget:** 25–60 min (min 20, max 90)
**Counteracts:** Orys-4 (productive friction), Orys-9 (forces verbalization)

**Canonical structure:**

```
🛠️ Roteiro de Simulação Procedimental
DR. DE.SOUZ.AI Orys™ — Camada 5 — Orys Learning Architecture™

ETAPA / OBSERVAÇÃO          [observation field]
1. OBSERVAR
   O que você vê no cenário/simulador/caso?

2. IDENTIFICAR
   Quais sinais ou variáveis são relevantes?

3. MECANISMO
   Qual conceito/teoria explica o que observei?

4. APLICAR
   Qual procedimento/abordagem cabe aqui?

5. JUSTIFICAR
   Por que esta escolha? Em 1 minuto, explique
   ao grupo a conexão entre observação e ação.

PRODUTO FINAL: Demonstração breve + justificativa
oral de até 1 minuto.

Citação: Kolb (1984); Schön (1983); de Souza (2025).
```

## Worksheet adaptation matrix by audience

| Level | Adaptation rules |
|---|---|
| **K-12 elementary** | Larger answer fields; simpler trigger questions; visual scaffolds (icons in headers); basic vocabulary |
| **K-12 middle/high** | Standard layout; reduce visual scaffold for high school; raise Bloom verb |
| **Undergraduate** | Tabular dense format; citation expectation in answers; paragraph-level depth |
| **Graduate/professional** | Minimal scaffold; theoretical grounding mandatory; evidence-based responses required |
| **Corporate L&D** | Workplace scenario framing; action-orientation; accountability columns |

Same template structure across all levels; the cognitive register changes.
